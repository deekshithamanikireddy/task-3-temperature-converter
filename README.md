# Temperature Converter Website

A simple web application that converts temperature between Celsius, Fahrenheit, and Kelvin.

## Features
- Input validation
- Celsius conversion
- Fahrenheit conversion
- Kelvin conversion
- Responsive design

## Technologies Used

- CSS3
- JavaScript
